Quick Start Code Examples

Get started with Vybez API in your favorite programming language.

Send a Message

const socket = io('https://yourdomain.com', {
  auth: {
    token: 'YOUR_JWT_TOKEN',
    client: 'api'
  }
});

socket.on('connect', () => {
  console.log('Connected to Vybez!');
  
  // Send a message
  socket.emit('chatMessage', {
    text: 'Hello from my app!',
    room: '#general'
  });
});

socket.on('chatMessage', (message) => {
  console.log('New message:', message);
});

Send a Message

import socketio

sio = socketio.Client()

@sio.event
def connect():
    print('Connected to Vybez!')
    sio.emit('chatMessage', {
        'text': 'Hello from Python!',
        'room': '#general'
    })

@sio.event
def chatMessage(data):
    print(f"New message: {data}")

sio.connect('https://yourdomain.com', auth={
    'token': 'YOUR_JWT_TOKEN',
    'client': 'api'
})

sio.wait()

Best Practices

Secure Your API Keys

Never expose API keys in client-side code. Keep them server-side and use environment variables.

Handle Rate Limits

Implement exponential backoff when you hit rate limits. Cache responses when possible.

Error Handling

Always handle Socket.IO disconnect events and implement reconnection logic with timeouts.

Log Everything

Keep detailed logs of API requests and responses for debugging and monitoring purposes.
