Frequently Asked Questions

Everything you need to know about Version 1.2

üìÖ When will Version 1.2 be released?

Version 1.2 is currently in development and is scheduled for release in Q1 2026. We'll be rolling out features gradually through beta testing before the official launch.

üîÑ Will my existing data be migrated?

Yes! All your existing rooms, messages, and user data will be automatically migrated to the new server structure. Your current rooms will become channels within a default server.

üíæ Can I still use the old room system?

The old room system will be fully integrated into the new server structure. You'll have all the same functionality plus powerful new organization features.

üé® How do custom emojis work?

Server admins can upload custom emoji images (PNG, GIF) up to 256KB. Members can then use these emojis in messages and reactions within that server. Each server can have up to 50 custom emojis.

üõ°Ô∏è How many roles can I create?

Each server can have up to 100 custom roles with granular permissions including manage server, kick/ban members, manage channels, send messages, and more.

‚úèÔ∏è Can I edit messages after a certain time?

Messages can be edited within 24 hours of sending. Edited messages will show an "(edited)" indicator and maintain a full edit history visible to moderators.

üîç How does server discovery work?

Server owners can list their servers in our public directory with tags, descriptions, and member counts. Users can search and filter to find communities that match their interests.
