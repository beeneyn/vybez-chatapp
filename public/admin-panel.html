Maintenance Mode

What is Maintenance Mode?

When maintenance mode is enabled, the /health endpoint will return a 503 status code, indicating that the system is under maintenance. This is useful for monitoring services and load balancers.

Note: This does not prevent users from accessing the application. It only affects the health endpoint status.
